# Introduction

`@yetric/audioplayer` is a headless TypeScript audio player for modern browsers.

It is built around one idea: make browser audio usable from vanilla JS, TypeScript, React, Angular, and Web Components without forcing playback logic into a framework.

## What it includes

- browser `HTMLAudioElement` engine
- normalized serializable player state
- built-in queue and repeat modes
- opt-in persistence
- opt-in progress reporting
- opt-in Media Session integration
- React adapter with provider and hooks
